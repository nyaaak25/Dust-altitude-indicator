# Dust-altitude-indicator
Data for JGR-Planets submission (Figures 3–6)

This repository contains the processed data used to generate Figures 3–6 in the manuscript:

"Dust Altitude Estimation on Mars Inferred from Two CO2 Bands Using Nadir Observation by OMEGA/Mars Express",
submitted to Journal of Geophysical Research: Planets.

Each text file includes gridded or sample-based data corresponding to one figure.
Columns and units are described in the file headers.
